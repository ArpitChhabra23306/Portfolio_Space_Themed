## Overview

A meticulously crafted developer portfolio that balances visual impact with performance and accessibility. Every interaction is intentional — from the terminal intro animation to the 3D-tilting project cards.

## Design Philosophy

- **Dark + Bold + Glassy**: Pure black base, frosted glass surfaces, cinematic orange accents
- **Motion with Purpose**: Every animation communicates hierarchy or state change
- **Content-First**: Static MDX/JSON content with zero runtime CMS overhead
- **Accessible by Default**: WCAG AA compliant, reduced-motion support, semantic HTML throughout
