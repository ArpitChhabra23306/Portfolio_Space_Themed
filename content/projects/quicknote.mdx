## Overview

QuickNote AI transforms note-taking from passive text capture into an intelligent knowledge management system. It understands context, surfaces connections between notes, and helps users retrieve information effortlessly.

## Key Features

- **Smart Summarization**: One-click AI summaries for long notes and lecture transcripts
- **Auto-Tagging**: NLP-based tag suggestions that improve organization without manual effort
- **Semantic Search**: Vector-based search using Pinecone that finds notes by meaning, not just keywords
- **Cross-References**: Automatic linking between related notes based on content similarity
- **Export**: PDF, Markdown, and Notion export formats
