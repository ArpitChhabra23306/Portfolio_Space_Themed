## Overview

DevFolio is an open-source portfolio generator designed specifically for developers. It supports MDX content, theme customization, and automatic deployment to Vercel.

## Features

- MDX-based content management
- Multiple theme presets
- Automatic SEO optimization
- One-click Vercel deployment
