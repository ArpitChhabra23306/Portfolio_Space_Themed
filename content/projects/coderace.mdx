## Overview

CodeRace gamifies DSA practice by turning it into head-to-head coding races. Players are matched by skill level and compete to solve algorithmic problems in real-time.

## Architecture

- **Real-time Engine**: Socket.io rooms for live race synchronization and spectator mode
- **Code Execution**: Sandboxed execution via Judge0 API with Docker containers
- **Matchmaking**: ELO-based rating system with skill-band queuing
- **Anti-Cheat**: Server-side solution validation with plagiarism detection

## Technical Challenges

- Sub-200ms latency for keystroke sync across players
- Fair timing guarantees despite network jitter
- Sandboxed code execution with resource limits
- Horizontal scaling with Redis-backed Socket.io adapter
