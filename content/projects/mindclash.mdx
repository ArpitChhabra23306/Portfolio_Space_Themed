## Overview

MindClash reimagines online debates by introducing AI-powered moderation, real-time scoring, and structured argumentation. The platform supports 1v1 and panel debates with live audience participation.

## Architecture

The system uses a microservices-inspired architecture with:
- **Frontend**: React SPA with real-time state sync via Socket.io
- **Backend**: Express API server with Redis pub/sub for horizontal scaling
- **AI Layer**: Gemini API integration for argument analysis and scoring
- **Database**: MongoDB for persistent data, Redis for session state and caching

## Key Features

- Real-time debate rooms with WebSocket communication
- AI moderation that flags logical fallacies and off-topic arguments
- Live scoring algorithm based on argument strength, evidence, and rebuttals
- Audience voting with real-time poll visualization
- Debate replay and highlight generation
