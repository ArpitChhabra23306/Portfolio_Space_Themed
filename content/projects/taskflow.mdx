## Overview

TaskFlow is a lightweight kanban-style project management tool built for small teams who want speed over features. Real-time sync ensures everyone sees changes instantly.

## Technical Highlights

- WebSocket-based real-time collaboration
- Optimistic UI updates with conflict resolution
- Docker-based deployment with PostgreSQL
- Drag-and-drop with keyboard accessibility
