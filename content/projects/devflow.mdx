## Overview

DevFlow is a developer productivity platform that brings together scattered development metrics into a single, beautiful dashboard. It helps individual developers and teams understand their coding patterns and improve workflow efficiency.

## Technical Highlights

- **GitHub Integration**: OAuth-based GitHub App with webhook listeners for real-time activity tracking
- **Analytics Engine**: Custom aggregation pipeline computing PR cycle time, review velocity, and contribution patterns
- **Widget System**: Drag-and-drop dashboard customization with persistent layout storage
- **Performance**: ISR for dashboard pages, WebSocket for live updates, Redis caching for API responses
